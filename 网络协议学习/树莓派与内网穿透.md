# 树莓派与内网穿透

```java
[Huawei]dis ip interface brief // 查看端口对应ip
```



配置好各个端口的网关，R1设置静态路由，添加6段和64段

```css
ip route-static 6.6.6.0 24 64.1.1.10
```
