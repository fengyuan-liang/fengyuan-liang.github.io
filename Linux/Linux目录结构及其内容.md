# Linux目录结构及其内容

在早期的`UNIX`系统中，关于某个文件应该放在哪个目录下的问题，各个发行版都有自己的观点。为了避免产生混乱，在`Linux`面世不久后就开始了对`Linux文件系统`的标准活动。Linux文件系统标准`FSSTND`于`1994年`推出，之后`FSSTND`又扩大到了`UNIX`系统，形成了文件系统层次标准`Filesystem Hierarchy Standard,FHS`。
